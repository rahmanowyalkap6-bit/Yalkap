Общее описание

Этот проект — это простая ферма, созданная на JavaScript с использованием ООП (объектно-ориентированного программирования). Пользователь может взаимодействовать с клетками поля, используя разные инструменты — лопату, семена и ведро. Каждая клетка может быть землёй, водой или содержать растение. Растения растут или умирают в зависимости от уровня влажности.

Структура HTML

HTML состоит из трёх основных частей:

Заголовок (head) Здесь находятся название страницы и CSS-стили, которые отвечают за внешний вид элементов: сетки, кнопок и панелей.

Боковая панель (sidebar) Содержит:

выбор инструментов: Лопата , Сеять , Ведро

выбор типа семян:

“Болотник”

“Картошка”

“Кактус”

правила и элементы управления (Сброс, Пауза).

состояние ведра (пустое или с водой).

Игровое поле (grid) Сетка размером 12×12 клеток. Каждая клетка представлена объектом класса Cell.

JAVASCRIPT — Объектно-ориентированная структура

Код написан в парадигме ООП. Основные классы:

class Cell

Представляет каждую клетку поля. Содержит свойства:

x, y — координаты

isSoil — является ли землёй

isWaterBlock — содержит ли воду

moisture — уровень влажности

plant — объект растения (если есть)

Методы:

toggleSoil() — перекопать землю (лопатой)

toggleWater() — добавить или убрать воду

calcMoisture(grid) — вычислить влажность на основе соседних водных клеток

tick(grid) — обновить состояние растения

moistureColor() — определить цвет клетки в зависимости от влажности (от сухого к влажному)

class Plant

Базовый класс для всех растений. Свойства:

growth — уровень роста

alive — живо ли растение

age — возраст

minMoisture, maxMoisture — диапазон подходящей влажности

emoji — визуальное представление растения

growthRate — скорость роста

Методы:

update(moisture) — обновляет состояние в зависимости от влажности

view() — возвращает HTML-код растения (размер эмодзи зависит от роста)

class BogPlant, class Potato, class Cactus

Три типа растений:

BogPlant (Болотник) — любит влажную почву (0.5–1.0)

Potato (Картошка) — средняя влажность (0.25–0.7)

Cactus (Кактус) — сухая почва (0.0–0.3)

Каждый класс наследует (extends) базовый класс Plant.

class Grid

Отвечает за всё поле. Создаёт клетки, управляет ими и отображает их на экране. Также реагирует на клики пользователя и каждую секунду обновляет состояние растений.

Методы:

createDom() — создаёт HTML-сетку

getCell(x, y) — возвращает клетку по координатам

tick() — обновляет влажность и растения

render() — отрисовывает клетки, воду и растения

reset() — сбрасывает поле и случайно добавляет несколько водных клеток

Инструменты (Toolbar)

Пользователь может выбрать один из трёх инструментов:

Лопата (Лопата) — копать или засыпать клетку

Сеять (Сеять) — посадить выбранное растение

Ведро (Ведро) — набрать или вылить воду

Если ведро полное — выливает воду, если пустое — набирает воду из источника.

Цикл симуляции

Функция setInterval обновляет состояние каждые 1 секунду:

пересчитывается влажность

обновляется рост растений

перерисовывается интерфейс

Симуляцию можно приостановить кнопкой «Пауза».

Визуальная логика

Цвет клетки меняется от сухого (светло-жёлтый) к влажному (тёмно-коричневому)

Вода обозначается

Растения отображаются эмодзи, увеличиваясь по мере роста

Итог

Этот проект — интерактивная мини-ферма на JavaScript, реализованная с помощью ООП. Пользователь может:

копать землю,

сажать растения,

добавлять воду,

наблюдать за ростом растений в зависимости от влажности.

Каждое растение живёт только в своём диапазоне влажности, и если условия неподходящие — оно погибает. Игра работает полностью в браузере и не требует внешних библиотек.
